2019年11月15日 10:09
linux
  1. apt install zsh
  2. git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh
  3. cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
  4. 默认启动 zsh
    chsh -s /bin/zsh

编辑配置文件

vim ~/.zshrc
  • 禁用自动更新
    DISABLE_AUTO_UPDATE="true"
  • 通配符的Bug
    find . -name *.h， *号匹配的搜索会有bug
    添加 setopt no_nomatch
  • 执行
    source ~/.zshrc
