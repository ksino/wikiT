2. Symbol 元件

Symbol的库是一个lib的扩展名的文件
这个库中的Symbol都在lib文件里面

新增元件

  • Symbol Editor
  • Create new Symbol
  • Select Symbol Library
    选择要保存的库
    名称以及符号
  • 一般会提供对应的元件图，按照元件图进行绘制

技巧

  • 修改坐标等等数据，可以直接在数据后面添加符号以及数字自动计算
    +号，或者-减号
    2000+100
  • 修改多个pin编号、坐标
    Edit -> Pin table
    先在excel准备好复制的数字，然后在pin table中选择，复制
  • 修改多个pin的管脚长度
    先修改一个pin后，右键这个pin
    Global -> push pin size to others
  • 填充颜色
    编辑属性，Fill style
    rectangle 矩形图形才能编辑

4. Footprint 封装

footprint的库是在以.pretty的文件夹中，如m200.pretty
文件夹里面的footprint是独立的单个文件

新增footprint

  • FootPrint Editor -> new footprint
    先输入名称，保存的时候选择相应的库

技巧

  • 添加多个过孔或者贴片
    先添加一个，然后右键create array（Ctrl + T）

PCB

  • pcb的各个层对应不同功能
    pcb的框架需要在Edge.cut
    正面连线F.cu
    背面连线B.cu
    过孔，从正面连线到背面，参考图片
