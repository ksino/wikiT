换行

通常段落的处理会导致单个换行符被忽略，且连续两个换行符被解释为一个段落的结束 这种行为不太方便处理包含了换行符的题材 - 例如，诗歌。您可以标记一个包含分行符号的区块，像这样：

"""
This is a line
and this is a new line
while this is yet another line
and this is the final one
apart from this one
"""

显示结果：

This is a line
and this is a new line
while this is yet another line
and this is the final one
apart from this one

... HTML 標記：

<p>This is a line<br>and this is a new line<br>while this is yet another line<br>and this is the final one<br>apart from this one<br></p>
