Wiki语法

WikiText的一个关键的功能，链接到条目或其他网站

手动链接

  • [[条目名称]]或者[[显示名称|条目名称]]
    Wiki语法或者超级好用的语法

驼峰式链接

  • 条目名称符合驼峰式规则，只输入条目名无需中括号就会产生链接，例如：CamelCase
  • 输入前置符号~，即可不产生链接

* ~HelloThere is not a link
* ~http://google.com/ is not a link

显示结果：

  • HelloThere is not a link
  • http://google.com/ is not a link

... HTML 標記：

<ul><li>HelloThere is not a link</li><li>http://google.com/ is not a link</li></ul>
