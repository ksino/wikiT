串口软件连接hmi屏幕

  • 串口软件必需加上结束符FF FF FF
  • 使用软件连接对应的com端口
    eaglecom
    page main \FF \FF \FF
    sscom
    点击扩展，勾选hex，输入 FF FF FF
    发送一个命令后，点击FF FF FF对应的右边标签

指令集

  • 跳转
    page 0
    page main
  • 存储变量
    wepo "abcd",10
    page main
  • 数字与字符串互转

covx 

covx att1,att2,lenth,format

att1:源变量

att2:目标变量

lenth:字符串的长度(0为自动长度，非0为固定长度)

format:申明数值类型(0-数字;1-货币;2-Hex)
