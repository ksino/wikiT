Wiki语法

多行引用

<<<
This is a block quoted paragraph
written in English
<<<

显示结果：

This is a block quoted paragraph written in English

... HTML 標記：

<blockquote class="tc-quote"><p>This is a block quoted paragraph
written in English
</p></blockquote>

引用

  • 引用可加入到引言区块：

<<<
相看两不厌，只有敬亭山。
<<< 李白

显示结果：

相看两不厌，只有敬亭山。

李白

... HTML 標記：

<blockquote class="tc-quote"><p>相看两不厌，只有敬亭山。
</p><cite>李白</cite></blockquote>
  • CSS可以加入到引言，例如大引号样式。

<<<.tc-big-quote
相看两不厌，只有敬亭山。
<<< 李白

显示结果：

相看两不厌，只有敬亭山。

李白

... HTML 標記：

<blockquote class="tc-quote tc-big-quote"><p>相看两不厌，只有敬亭山。
</p><cite>李白</cite></blockquote>
