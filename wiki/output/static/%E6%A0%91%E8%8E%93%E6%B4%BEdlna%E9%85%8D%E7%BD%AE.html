树莓派dlna配置

2020年04月08日 21:58
linux raspberry

配置

  • 配置访问路径
    sudo vim /etc/minidlna.conf
user=root
media_dir=V,/media/pi/EasyU/
  • minidlna挂载的硬盘中的不显示视频
    sudo vim /etc/default/minidlna
USER="root"
DAEMON_OPTS=""

更新资源

新增文件，执行以下命令
  • 强行更改配置
    sudo service minidlna force-reload
  • 手动更新minidlna数据库并重启服务
    sudo minidlnad -R
    sudo systemctl restart minidlna.service
