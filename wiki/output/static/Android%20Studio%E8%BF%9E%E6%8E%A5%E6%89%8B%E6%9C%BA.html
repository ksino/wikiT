ADB环境配置

  • 查看自己Android Studio配置的sdk路径
    Android Studio –>File –>Project Structure –> SDK Location
    例如这里是D:\Program\Android\Sdk
  • 新建环境变量
    win7配置环境变量
    变量名：ANDROID_HOME
    变量值：D:\Program\Android\Sdk\platform-tools
  • 添加环境变量到path
    编辑path，直接添加路径或者%ANDROID_HOME%
  • 命令行输入adb，检测是否配置成功

下载Google USB drive

  • Android Studio –> 选择file –>Settings –> 搜索sdk –> 点击Android SDK
  • 选择SDK Tools –> 勾选Google USB driver
  • 点击Apply
